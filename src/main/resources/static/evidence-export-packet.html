Packet Purpose

This is a human-readable handoff packet template. It gathers existing evidence paths, verification commands, dashboard links, CI dry-run artifact metadata, governance references, and not-proven boundaries into one reviewer-facing surface.

It does not export secrets, generate files, create a release artifact, publish anything, sign anything, mutate cloud resources, mutate GitHub settings, or certify production use.
