9. Unavailable-state handling

      

      

        

          If an endpoint returns 404, 405, or a section is not supported by the current API, this cockpit marks
          that section as Not available in current API and keeps the rest of the demo usable.
          It does not fabricate allocation, routing, load-shedding, or remediation behavior.
