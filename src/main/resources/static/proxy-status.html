Demo curl commands

mvn -q -DskipTests compile
java -cp target/classes com.richmond423.loadbalancerpro.demo.ProxyDemoFixtureLauncher --mode round-robin
java -cp target/classes com.richmond423.loadbalancerpro.demo.ProxyDemoFixtureLauncher --mode weighted-round-robin
java -cp target/classes com.richmond423.loadbalancerpro.demo.ProxyDemoFixtureLauncher --mode failover
powershell -ExecutionPolicy Bypass -File .\scripts\proxy-demo.ps1 -Mode round-robin
powershell -ExecutionPolicy Bypass -File .\scripts\proxy-demo.ps1 -Mode weighted-round-robin
powershell -ExecutionPolicy Bypass -File .\scripts\proxy-demo.ps1 -Mode failover
bash scripts/proxy-demo.sh --mode round-robin
bash scripts/proxy-demo.sh --mode weighted-round-robin
bash scripts/proxy-demo.sh --mode failover

Safety / limitations

  • Read-only UI backed only by same-origin GET /api/proxy/status.
  • No backend writes, no reset controls, and no mutation controls.
  • No cloud mutation and no external services.
  • No browser storage; live refresh state stays in memory for this page session.
  • Lightweight local operator view, not production monitoring.
